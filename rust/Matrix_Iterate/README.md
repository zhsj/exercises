# Lab06

## 运行

```bash
# install rust
curl -L https://static.rust-lang.org/rustup.sh | sudo sh

# install on windows
# https://static.rust-lang.org/dist/rust-1.0.0-x86_64-pc-windows-gnu.exe

# run
cargo run
```

### 在线运行

可以不安装 Rust 程序，在网页上运行代码并得到结果。
